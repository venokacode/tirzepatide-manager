# Tirzepatide 注射管理系统

一个专业的 Tirzepatide（替尔泊肽）注射管理 Web 应用，帮助用户追踪和管理每周注射计划。

## 功能特点

### 📊 核心功能
- **智能初始化** - 支持新开始和继续已有疗程
- **剂量管理** - 4针周期自动追踪（2.5mg - 15mg）
- **注射提醒** - 浏览器通知，多时间点提醒
- **风险检查** - 注射前4项强制安全确认

### 💉 注射管理
- **部位轮换** - 智能检查28天间隔规则
- **实时状态** - 显示每个部位的可用性
- **历史记录** - 完整的注射日志和副作用追踪
- **超期处理** - 智能提示补打或跳过

### ⚙️ 设置功能
- **联系人管理** - 添加家人/朋友接收提醒
- **通知设置** - 自定义提醒方式
- **数据导出** - JSON格式备份
- **数据清除** - 双重确认保护

### 📖 使用指南
- 完整的注射步骤说明
- 储存要求详解
- 常见副作用列表
- 禁忌症提醒
- 锐器处理指南

## 技术栈

- **纯前端** - HTML + JavaScript + CSS
- **UI 框架** - Tailwind CSS
- **数据存储** - LocalStorage（本地存储）
- **无需服务器** - 完全离线可用

## 使用方法

### 在线访问
直接访问 GitHub Pages 链接即可使用

### 本地使用
1. 下载或克隆本仓库代码
2. 直接用浏览器打开 `index.html` 即可体验全部功能
3. 所有数据都会保存在浏览器本地的 LocalStorage 中

### 本地预览（推荐）
在开发或调试阶段，可通过任何静态服务器工具查看页面效果：

```bash
npx serve .
# 或者使用 Python 内置服务器
python3 -m http.server 4173
```

浏览器访问终端提示的地址（例如 <http://localhost:4173>）即可查看最新改动，确保样式与通知功能在真实环境中生效。

### 移动端
- 在手机浏览器中打开
- 点击浏览器菜单 → "添加到主屏幕"
- 像原生应用一样使用

## 数据安全

- ✅ 所有数据仅存储在您的浏览器本地
- ✅ 不会上传到任何服务器
- ✅ 支持数据导出备份
- ⚠️ 清除浏览器数据会丢失记录，请定期导出
